By default, you are now in a "default" project. If the tool will be used by several teams, then it is better to create your own project by clicking on the button in the lower left corner.

Configure InflixDB
The tool connects to influxdb to get test data that is used for the "Test" page and reports. To add a new influxdb configuration, go to the "Integrations" page.
Prev Next
Configure Grafana
Grafana is used to visualize graphs. It is important to note that grafana must have "grafana-image-renderer" plugin or "grafana-image-renderer" docker container connected. To add new Grafana config go to the "Integrations" page.
Prev Next
Configure Output
The tool can export reports to azure wiki, confluence wiki and confluence jira. To add new target destination, go to "Integrations" page.
Prev Next
Add graphs
Then you need to configure which graphs from grfana can be rendered. To add a new graph, go to the flow page.

REPORT
Configure reporting flow
When all the integrations are set up, you need to create a report flow. Where you need to specify which influxdb, grafana should be used and to which destination the report should be saved. To configure this, go to the flow configuration page.
Prev Next
Configure reporting template
To configure the structure of the report, you need to create a template on the flow configuration page. The syntax depends on the target destination. In addition, you can use various variables that will be replaced with values.
